# CareerChat

CareerChat — это анонимное веб-приложение для общения между людьми из разных профессий в формате случайного чата.

Пользователь выбирает свою профессию и (при желании) указывает, с кем хочет поговорить. После этого система подбирает собеседника и соединяет их в анонимном чате 1 на 1.

---

## 🚀 Функциональность

* 🔍 Поиск собеседника по профессии
* 💬 Чат в реальном времени (WebSocket)
* ⌨️ Индикатор "собеседник печатает"
* 🔄 Кнопка "Следующий собеседник"
* ❌ Завершение чата с уведомлением
* 🧠 Автоподсказки профессий
* ✅ Валидация ввода
* 🔒 Полная анонимность (без аккаунтов)

## ⚙️ Архитектура

Проект разделён на frontend и backend.

Frontend:

* `app.js` — точка входа приложения 
* `ui.js` — управление экранами
* `chat.js` — логика чата
* `socket.js` — работа с WebSocket
* `professions.js` — автоподсказки
* `validation.js` — проверка формы

Backend:

* `server.js` — запуск сервера 
* `matchmaking.js` — логика подбора собеседников 
* `routes/professions.js` — API профессий

---

## 🔌 Как работает приложение

1. Пользователь вводит профессию
2. Отправляется запрос `join_queue` через Socket.IO
3. Сервер подбирает пару пользователей
4. Создаётся комната (`room`)
5. Пользователи общаются в реальном времени


## 💡 Идея проекта

CareerChat решает проблему отсутствия живого общения между новичками и специалистами.

Многие люди:

* не знают, с чего начать карьеру
* не могут задать вопросы специалистам
* получают только общую информацию из интернета

CareerChat позволяет быстро и анонимно получить реальный опыт от человека из профессии.
